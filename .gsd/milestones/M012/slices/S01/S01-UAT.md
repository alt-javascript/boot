# S01: DotEnvParser + ProfileConfigLoader .env integration — UAT

**Milestone:** M012
**Written:** 2026-03-25T19:41:33.559Z

## UAT — .env File Support\n\n### Setup\n\nCreate `config/application.env` alongside your existing `config/application.json`:\n\n```env\n# config/application.env\nDATABASE_URL=postgres://localhost:5432/mydb\nAPP_PORT=5000\nSECRET_KEY=\"my secret value\"\n```\n\n### Test 1: Basic discovery and relaxed binding\n\n```js\nimport { ConfigFactory } from '@alt-javascript/config';\nconst config = ConfigFactory.loadConfig();\n\nconsole.log(config.get('DATABASE_URL'));   // postgres://localhost:5432/mydb\nconsole.log(config.get('database.url'));   // postgres://localhost:5432/mydb (relaxed binding)\nconsole.log(config.get('app.port'));       // 5000\nconsole.log(config.get('secret.key'));     // my secret value\n```\n\n### Test 2: process.env wins\n\n```sh\nDATABASE_URL=postgres://prod-server/prod node -e \"\nimport { ConfigFactory } from '@alt-javascript/config';\nconst c = ConfigFactory.loadConfig();\nconsole.log(c.get('DATABASE_URL')); // postgres://prod-server/prod\n\"\n```\n\n### Test 3: Profile-specific .env\n\nCreate `config/application-dev.env`:\n\n```env\nDATABASE_URL=postgres://localhost:5432/devdb\n```\n\n```sh\nNODE_ACTIVE_PROFILES=dev node -e \"\nimport { ConfigFactory } from '@alt-javascript/config';\nconst c = ConfigFactory.loadConfig();\nconsole.log(c.get('database.url')); // postgres://localhost:5432/devdb\n\"\n```
